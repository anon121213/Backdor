Программа выполняет команды в командной строке зараженного компьютера, для работы программы требуется ввести в код как клиента (вируса) так и сервера (программы с которой будет происходить управление) ip адрес компьютера с которого будет происходить ввод. Так же для работы программы требуется чтобы как зараженный компьютер так и компьютер с которого будет производиться управление были подключены к одой сети.
При запуске вируса на компьютере она автоматически запишется в автозагрузку компьютера. Сам сервер такой возможностью не обладает. При запуске сервера высветиться окно командной строки с надписью: ”Waiting for connection...” 
В случае подключения будет написано: ‘’Connection from (и ip адресс компьютера к которобы было произведено подключение) ’’
Программа обязует наличие Python на компьютере с которого будет происходить управление.
В случае необходимости скрыть программу с зараженного компьютера можно ее собрать в .exe файл при помощи pyinstaller
Для этого следует его установить с помощью консольной команды 

Pip install pyinstaller 

Далее в консоли следует перейти в директорию с файлом клиентом при помощи команды ‘’cd’’ и ввести команду для сборки .py  файла в .exe 

Pyinstaller client.py --onefile --noconsole  

По умолчанию программа запускается с правами обычного пользователя.     
Далее можно управлять командной строкой зараженного компьютера.
